# Vim学习笔记


### vim的三种模式

- **normal**  普通模式
- **insert**  插入模式
- **virtual**  虚拟模式  

---

> normal --> insert 模式的几种基本操作  
>- i insert before cursor
>- I insert before line 
>- a append after cursor
>- A append after line
>- o open a new line below
>- O open a new line above
